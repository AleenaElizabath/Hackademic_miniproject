Password Creation — Easy

Core idea: Teach learners how to create strong, memorable passwords using rules, entropy feedback, and safe practices.

Difficulty

Easy — guided steps and examples.

Scoring

Points based on password strength metrics (length, character variety, absence of common words). Bonus for mnemonic techniques.

What you'll learn

  • Why long, unique passwords matter
  • How to combine words and symbols to increase entropy
  • Safe practices: avoid reusing passwords and using personal info
